Welcome

Welcome to Drzzle's design and development system. Here you will find guidelines on how to develop new droplets and themes, as well as find code snippets on how to initiate and impliment all existing things.

Themes in Picazzo exist merely to provide default styling for all droplets and components. Most styling can be overwritten using the Drzzle page editor.

Use Picazzo in Your Project

NPM

npm install picazzo --save

You can then access all assets from your node_modules folder. 

node_modules/
..dist/
....css/
......themes/
....js/
......droplets/
......js files
