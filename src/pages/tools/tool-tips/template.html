Options

Options are set using HTML attributes. See demo. 

// this will be the content in the tool tip
data-tooltip-content="My Content Here"
// directional options
data-tooltip-position="top"
data-tooltip-position="bottom"
data-tooltip-position="left"
data-tooltip-position="right"
