Initializing Plugin

$('.drzFilter-grid').drzFilterGrid({
  feed: [
    {
      created: '2020-06-25T19:31:14.835Z',
      description: 'First Item description here.',
      name: 'First Item',
      categories: ['movie'],
    },
    {
      created: '2020-06-24T19:31:14.835Z',
      description: 'Second Item description here.',
      name: 'Second Item',
      categories: ['movie', 'music'],
    },
  ],
  output(data) {
    // this is the HTML that will render out for each
    // feed item
    return `<div>${data.name}</div>`;
  },
  // the search input will look through the feed item
  // keys of your choice
  searchKeys: ['name', 'description'],
  // optional after each feed item renders
  onRender($item) {
    // add events to $item
  },
  // optional if you need global filtering. These need
  // to correspond to the feed item categories
  filters: ['movie', 'music'],
  // optional if you need pagination. The number indicates
  // the number of items you want per set
  pagination: 4,
});

Lazy Load Images

To lazy load any assets in the output hook, you will need to use the data-asset attribute. When the item comes into view, the image's url will auto set the src initializing the load. See example below. 

output(data) {
  return `
    <div>
      <img data-asset="${data.someImage}" />
    </div>
  `;
}

Destroy Plugin

const $droplet = $('.drzFilter-grid');
$droplet.drzFilterGrid.destroy($droplet);
