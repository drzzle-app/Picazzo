Initializing Plugin

This plugin is very much tied to the Drzzle e-commerce services. The API parameter shouldn't be changed as it handles shipping, inventory, purchase and discount calls. 

$('.drzSlideCheckout').drzSlideCheckout({
  siteId: 'drzzle-site-id',
  taxPercent: 8.25,
  api: 'https://api.drzzle.app',
  currency: {
    type: 'USD',
    symbol: '$',
  },
});
Droplets

This plugin can mark product droplets "out of stock" by adding a class and message if it's indeed out of stock. To have this feature on a product droplet you'll need to add 2 attributes (data-no-stock, data-product-droplet) to the HTML. See the following example. 

<div class="drzProduct-feature" data-product-droplet="product-id-here">
  ...
  <!-- add to any "add-to-cart" buttons -->
  <a href="#"
    class="drzProduct-feature-buyBtn"
    data-no-stock="product-id-here"
    name="add-to-cart">
    Add to Cart
  </a>
</div>
