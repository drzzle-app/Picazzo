Javascript

Initializing Plugin

$('.filter-controls').sortFilter();

All plugin options are set through HTML attributes.

Search Options

// This attribute needs to be added to the input doing the searching
data-filter-search="true"

Sort Filter Options

See sort filter demo for how this is implementation.

// Sort by date (add to filter buttons)
data-filter-sort="oldest"
data-filter-sort="newest"
// Then on content to be sorted, add date
data-filter-sort="12/01/20"

// Sort by category (on filter button)
data-filter-category="anynamehere"
// Then on content to be sorted, match the attr
data-filter-category="anynamehere"
// To add a reset button
data-filter-reset="true"
