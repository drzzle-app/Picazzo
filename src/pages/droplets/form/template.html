Initializing Plugin

This plugin is to cover front end validation. 
$('.drzForm').drzFormValidate(function() {
  event.preventDefault();
  // code to run after passed validation
});

Options

Options are set using the input HTML attributes. See above demo HTML. 

// set input required
data-validator="required"
// set a custom error message
data-validator-msg="my custom error message"
// min and max range for number inputs
data-validator-min="1"
data-validator-max="10"
// required strictly a number value
data-validator-value="number"
// input must match your custom regex value
data-validator-regex="^[A-Z ]+$"
// custom regex failure message
data-validator-regex-msg="Only letters allowed."

Validating Checkbox and Radio Groups

In addition to initializing the validation plugin, you will also need to add a corresponding class to the field parent of a group of checkboxes (checkbox-group) or radio buttons (radio-group). 

{{ checkboxValidate() }}
{{ radioValidate() }}

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
