Options

This plugin can take in multiple end times. Options are set using HTML attributes. You will need to pass in an array of end times in the following JSON format. See above demo HTML. 

// countdown end time
"end": "Mon May 29 2017 09:30:00 GMT-0500 (CDT)"
// delay next countdown (if multiple) for this amount of time.
"buffer": "1 hour"
// countdown end message.
"endMessage": "Ended 1st Countdown!"
// recurring is currently only available for weekly only. The default value is false.
"recurring": "every week"
