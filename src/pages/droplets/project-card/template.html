Initializing Plugin

This droplet uses the Load More plugin. View that page for further documentation. Below describes the code needed for this droplet's particular options. 

$('.drzProject-cards').drzLoadMore({
  visible: 4, // optional. number of items to show at a time
  assets: 'bg',
  classes: {
    show: 'drzProject-card-show',
    img: 'drzProject-card-image',
    footer: 'drzProject-cards-footer',
    footerHidden: 'drzProject-cards-footerHidden',
    item: 'drzProject-card',
    loadMore: 'drzProject-cards-more',
  },
});

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
