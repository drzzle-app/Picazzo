Initializing Plugin

$('.drzContentSlider').drzContentSlider();

Options

Options for this plugins are set using HTML attributes. 

// position of bullets
data-slider-bullets="bottom" // default
data-slider-bullets="right"
data-slider-bullets="left"
// controls
data-slider-controls="show" // default
data-slider-controls="hide"
// transition effects
data-slider-effect="slide" // default
data-slider-effect="fade"
// delay in seconds
data-slider-delay="4" // default
data-slider-delay="none" // to disable auto sliding

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
