Initializing Plugin

This droplet uses the Load More plugin. View that page for further documentation. Below describes the code needed for this droplet's particular options. 

$('.drzMediaItem-list').drzLoadMore({
  visible: 2, // optional. number of items to show at a time
  assets: 'src',
  classes: {
    show: 'drzMediaItem-item-show',
    img: 'drzMediaItem-item-img',
    footer: 'drzMediaItem-list-footer',
    footerHidden: 'drzMediaItem-list-footerHidden',
    item: 'drzMediaItem-item',
    loadMore: 'drzMediaItem-list-more',
  },
});

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
