Initializing Plugin

This droplet is best done on a single line basis. Multiple lines are currently not supported. 

$('.drzText-swapper').drzTextSwapper();

Options

Options for this plugin are set using HTML attributes. They need to be added to each group of text items that will be animated. 

// the amount of time in milliseconds until swapping words. default is 3000.
data-swapper-pause="3000"
// swap effect. 3 options: fade, rise or zoom
data-swapper-effect="fade"
// animation transition duration in milliseconds. default is 500.
data-swapper-duration="500"

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
