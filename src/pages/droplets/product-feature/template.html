Options

Options for this plugin are set using HTML attributes and within js paramaters. 

// zoom on hover option
data-image-zoom="true" // default is false

On plugin initialization, you can pass some options into the plugin. 

$('.drzProduct-feature').drzProductFeature({
  onCartAdd() => {
    // callback when clicking cart button
  },
  // this will override the quantity counter limits
  overrides: {
    quantity: {
      max: 5,
      step: 3,
    },
  },
});

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}

Options Plugin

This droplet also uses another plugin that can be used by any product droplet that may need it in the future. This plugin is for droplets that may have any options or add and remove count buttons in it like this droplet does. The plugin gathers the selected counts and options attached and passes it to the shopping cart when users add items to their cart.

The following attributes would be needed in the droplet HTML:

data-product-option="option-id" - On any select options for the product
name="add-to-cart" - On cart button
name="product-count" - On any element displaying the numeric count desired
name="add-product-count" - On any button that will be adding to the count
name="remove-product-count" - On any button that will be removing to the count 

<div class="myDroplet">
  <!-- optional select options -->
  <select>
    <option data-product-option="option-id">Large</option>
    <option data-product-option="option-id">Small</option>
  </select>
  <!-- optional product count editing -->
  <button name="add-product-count"> + </button>
  <span name="product-count">3</span>
  <button name="remove-product-count"> - </button>
  <!-- add to cart button -->
  <button name="add-to-cart">Add to Cart</button>
</div>
$('.myDroplet').drzProductOptions();
