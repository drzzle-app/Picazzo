For accordions, the a 'drzAccordion-section-title' href attribute must match the 'section-' class with the 'drzAccordion-section-content'. See below to initialize the plugin. 

$('.drzAccordion').drzAccordion();

Destroy Plugin

To detatch event listeners from the bounded element, call the .killEvents() method. This will come in handy for dynamically editing this pattern in the CMS. Try the "Test Kill" button, it should kill the demo events.
