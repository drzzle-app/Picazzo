Initializing Plugin

For accordions, the a 'drzAccordion-section-title' href attribute must match the 'section-' class with the 'drzAccordion-section-content'. 

$('.drzAccordion').drzAccordion();

Destroy Plugin

To destroy the plugin, use the .killEvents() method. This remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
