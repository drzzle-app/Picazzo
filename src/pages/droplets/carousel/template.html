Initializing Plugin

$('.drzCarousel-container').drzCarousel();

Options

Options for this plugins are set using HTML attributes. 

// number of visible images
data-carousel-visible="4" // default is 4
// delay in seconds for shift
data-carousel-delay="2" // default is 2
// show / hide controls
data-carousel-controls="show" // default
data-carousel-controls="hide"

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
