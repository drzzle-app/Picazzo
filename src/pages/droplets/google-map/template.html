Initializing Plugin

$('.drzMap').drzMap();

Options

Options are set using HTML attributes. You will need to pass in an array of map markers in JSON format. See above demo HTML. 

// custom color scheme to draw from
"baseColor": "#ff0000",
// if you want the color scheme to be drawn from bottom border color
"useBg": false
// map markers
"markers": [
  {
    "lat": 1111111,
    "long": 11111111,
    "address": '123 fake st, san antonio, tx',
    "title": "My Title",
    "markerImg": "/path/to/image.png"
  }
]

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
