Initializing Plugin

$('.drzBlog-list').drzLoadMore({
  visible: 4, // optional. number of blog items to show at a time
  assets: 'src',
  classes: {
    show: 'drzBlog-item-show',
    img: 'drzBlog-item-img',
    footer: 'drzBlog-list-footer',
    footerHidden: 'drzBlog-list-footerHidden',
    item: 'drzBlog-item',
    loadMore: 'drzBlog-list-more',
  },
});

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
