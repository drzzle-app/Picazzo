Class Options

Positional / Scrolling

You have an option to fix the nav to the top of the page or use a sticky nav with either of the following classes. The sticky nav will (as it sounds) stick to the top once you scroll past it on the page. The class will needed to be added to the "drzNav" div. 

drzNav-sticky
drzNav-fixed

Aligning Links

To align nav links right, left or center in desktop, choose between one of these classes. It needs to be added to the "drzNav" div. 

drzNav-links-right
drzNav-links-left
drzNav-links-center

Aligning Logo

To align the nav logo right or left, choose between one of these classes. It needs to be added to the "drzNav" div. 

drzNav-logo-right
drzNav-logo-left
Javascript

Initializing Plugin

$('.drzNav').drzNav();

Options

Options for this plugin are set using HTML attributes. This option is for the direction that the mobile slide out menu will appear from. 

data-nav-slide='left' // default
data-nav-slide='right'
data-nav-slide='center'

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}

See Plugin Source

Optional Search Plugin

The demo above uses the site search plugin. This plugin needs a separate initialization. 

const $nav = $('.drzNav');
$nav.find('.drzNav-search-btn').drzModal();
$nav.find('.drzModal-search-bar').drzSiteSearch();

Search Plugin Options

The search plugin takes in an option using an HTML attribute. It is the location of a JSON file with an array of pages to search from. This needs to be added to the search input element. See demo. 

data-search='{
"file": "js/app-search.json"
}'

Searchable JSON Format

The JSON file being searched needs to be in the following format. 

[
  {
    "pagetitle":"My Page Title",
    "href": "link-to-page.html",
    "metadata":"Searchable key words for this page"
  },
  {
    "pagetitle":"My Page 2 Title",
    "href": "link-to-page-2.html",
    "metadata":"Searchable key words for this page"
  }
]
