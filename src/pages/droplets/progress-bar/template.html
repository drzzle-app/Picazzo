Initializing Plugin

The progress bar will display the percentage of completion from 0 - 100% based off of the 2 values you pass. Both values can be numbers or strings. You can even include symbols like '$50' in order to display them on the bar. 

$('.drzProgressBar').drzProgressBar({
  progress: 50,
  total: 100
});

Destroy Plugin

To destroy the plugin, use the .destroy() method. This will remove all event listeners and added classes/attributes. 

{{ destroyMarkup() }}
