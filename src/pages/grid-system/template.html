Grid System

Columns are based on a 12 column grid system. It's not necessary, but rows are better laid out when inside a container div. Column elements need the "column" class followed by the "col-*" class.

Device Specific Columns

If you don't want to use the default columns, you can specify what device you'd like to use the fraction width for. The class format is as follows

.col-{device}-{fraction}

Device letters need to be in order. 

device = d (desktop), t (tablet), m (mobile)
fraction = 1 - 12 (as seen above)
